# Base de données avec MCP Web menu AR

## Instructions

**IMPORTANT** : Chaque fois que vous devez travailler avec la base de données, utilisez **exclusivement** le MCP "Web menu AR" pour effectuer toutes les opérations.

### Opérations couvertes

- **Modifications** : Utilisez le MCP pour modifier les données existantes
- **Ajouts** : Utilisez le MCP pour ajouter de nouvelles entrées
- **Mises à jour** : Utilisez le MCP pour mettre à jour les enregistrements
- **Requêtes** : Utilisez le MCP pour interroger la base de données
- **Suppressions** : Utilisez le MCP pour supprimer des données

### Quand utiliser cette skill

Cette skill doit être appliquée automatiquement lorsque :
- L'utilisateur demande de modifier des données dans la base de données
- L'utilisateur demande d'ajouter des données
- L'utilisateur demande de mettre à jour des informations
- L'utilisateur mentionne des opérations sur la base de données
- Vous devez interagir avec la base de données dans le contexte du projet Web-AR-Resto

### Principe

Ne pas utiliser d'autres méthodes ou outils pour interagir avec la base de données. Le MCP "Web menu AR" est l'interface standard et unique pour toutes les opérations de base de données dans ce projet.
